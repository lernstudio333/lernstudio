# Connect Sources

The **Connect Sources** feature lets admins link external documents (e.g. Google Docs) to Lernstudio so that flashcard content can be pushed in from the source directly.

---

## How it works

1. **Open the modal** — click your avatar in the top-right corner and choose **Connect Sources**.
2. **Copy the pairing code** — a 6-digit code appears automatically. Copy it with the copy button. The code expires in **5 minutes** and can only be used once.
3. **Paste into your Google Doc** — open the Lernstudio sidebar in your Google Doc and paste the code when prompted.
4. **Connection confirmed** — once the sidebar exchanges the code, your document appears in the **Connected Docs** table at the bottom of the modal.

---

## Connected Docs table

| Column | Description |
|--------|-------------|
| Doc Name | The name provided when the connection was set up |
| Doc ID | The unique identifier of the source document |
| Linked Account | The admin account that created the connection |
| Last Sync | When cards were last pushed from this document |

---

## FAQ

**The code expired before I could paste it.**
Close and reopen the modal — a fresh code is requested automatically each time.

**My document doesn't appear in the table after pairing.**
Reload the modal (close and reopen). The table refreshes on open.

**Can I connect the same document more than once?**
Yes — each pairing creates a separate session. Only the most recent sessions appear relevant in the table.
